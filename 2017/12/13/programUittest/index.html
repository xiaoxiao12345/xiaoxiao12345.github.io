自己项目的试验

出现的问题

根据django官方文档编写的程序出现的小问题，TypeError: view must be a callable or a list/tuple in the case of include().

可能官方文档也是有他自己的环境的，所以在我这里实现不了。遇到问题后想着是什么原因，个人想了一下可能是path的问题以及urlpatterns的问题，我就在api reference里看了一下url，发现url这个api会自动找urlpatterns并且urlpatterns必须是一个列表，url的api里也有提及到path，点进去看后明白path的参数的意义，并且试着试着就解决了问题。

代码区：
fruitStore/urls.py
from django.urls import include, path

urlpatterns = [
    path('', include('startFruitStore.urls')),
]

startFruitStore/urls.py
from django.urls import include, path
from . import views

urlpatterns =[
    path('',views.index, name='index')
]
