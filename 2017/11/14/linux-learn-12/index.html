第12章 学习shell script

1.請建立一支 script ，當你執行該 script 的時候，該 script 可以顯示： 1. 你目前的身份 (用 whoami ) 2. 你目前所在的目錄 (用 pwd)

#!bin/bash
whoami;pwd

运行：sh hello.sh

2.請自行建立一支程式，該程式可以用來計算『你還有幾天可以過生日』啊？

declare -i today_date=$(date +%s)
//这里要定义为整形才可以做减法，= 前后不能有空格 ，date +%s这个是date的写法，可以在date –help里查到

declare -i mybirthday=$(date +%Y1229)
//1229是我的生日

declare -i mybirthday_second=$(date –date=”$mytbirthday” +%s)
declare -i day=$[$mybirthday_second-$today_date]

declare -i day_1=$[$day/60/60/24]
echo $day_1

3.讓使用者輸入一個數字，程式可以由 1+2+3… 一直累加到使用者輸入的數字為止。
#!/usr/local/bin/python3.4

 # -*- coding: UTF-8 -*-

 num = input("please input your num:")

 i = 0

 total_sum = 0

 while i <int(num)+1:

     total_sum = total_sum + i

     i = i+1

 print(total_sum)
4.撰寫一支程式，他的作用是: 1.) 先查看一下 /root/test/logical 這個名稱是否存在； 2.) 若不存在，則建立一個檔案，使用 touch 來建立，建立完成後離開； 3.) 如果存在的話，判斷該名稱是否為檔案，若為檔案則將之刪除後建立一個目錄，檔名為 logical ，之後離開； 4.) 如果存在的話，而且該名稱為目錄，則移除此目錄！

&& 这里用到第十章只是 如果前一条正确才执行下一条。
|| 如果前一条不正确才执行下一条。
$？上一条命令是否正确，正确显示0，不正确显示非0

test有很多参数，-e文件名是否存在，-f 是不是文件，-z 测试是不是字符串，是字符串就为true。 -a 意思的and ，两个同时成立， -o or的意思。！ 否的意思

test ! -e /root/test/logical && test -d root/test && touch /root/test/logical && exit 0

test -e /root/test/logical && test -f /root/test/logical && cd /root/test/……. 类似的做法

5.我們知道 /etc/passwd 裡面以 : 來分隔，第一欄為帳號名稱。請寫一隻程式，可以將 /etc/passwd 的第一欄取出，而且每一欄都以一行字串『The 1 account is “root” 』來顯示，那個 1 表示行數。
