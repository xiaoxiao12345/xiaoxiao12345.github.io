第十章题目
情境模擬題一：由於 ~/.bash_history 僅能記錄指令，我想要在每次登出時都記錄時間，並將後續的指令 50 筆記錄下來， 可以如何處理？

显示隐藏文件 l.， 退出的操作是在隐藏文件 .bash_logout里面的。
在bash_logout里执行，将时间加到历史记录里 date >> ~/.bash_history，以及history 50 >> /.bash_history

简答题
1.在 Linux 上可以找到哪些 shell(舉出三個) ？那個檔案記錄可用的 shell ？而 Linux 預設的 shell 是？

kshell ,sh,cshell,档案记录用 etc/shells ，linux预设的shell是、bin/bash

2.你輸入一串指令之後，發現前面寫的一長串資料是錯的，你想要刪除游標所在處到最前面的指令串內容，應該如何處理？

[ctrl]+u/[ctrl]+k 分別是從游標處向前刪除指令串 ([ctrl]+u) 及向後刪除指令串 ([ctrl]+k)。(该组合键只是在一行上的)
[ctrl]+a/[ctrl]+e 分別是讓游標移動到整個指令串的最前面 ([ctrl]+a) 或最後面 ([ctrl]+e)。（ctrl+a 可以删除了该游标前面的所有的东西）（Ctrl +e 只是将下一行的内容再重现）

3.在 shell 環境下，有個提示字元 (prompt)，他可以修改嗎？要改什麼？預設的提示字元內容是？

可以修改，改ps1的表现形式，预设的提示字元为 ps1=’[\u@\h \W]\$’

4.如何顯示 HOME 這個環境變數？

环境变数是以大写字元表示区别与自定义的变数。 echo $HOME

5.如何得知目前的所有變數與環境變數的設定值

env 看所有环境变数，set看自定义变数.

6.我是否可以設定一個變數名稱為 3myhome

是的

7.在這樣的練習中『A=B』且『B=C』，若我下達『unset $A』，則取消的變數是 A 還是 B？

取消的是B，unset A 与unset $A的区别。

8.如何取消變數與命令別名的內容？

unset 变数名 unalias 命令别名

9.如何設定一個變數名稱為 name 內容為 It’s my name ？

name=”It’s my name”

10.bash 環境設定檔主要分為哪兩種類型的讀取？分別讀取哪些重要檔案？

source 设定档档名
cat 设定档
~/.bash_profile (login shell 才會讀)
~/.bashrc (non-login shell 會讀)
将文件放到~/bin下就类似于window 下的环境变量了。

11.CentOS 7.x 的 man page 的路徑設定檔案？

/etc/man_db.conf

12.試說明 ‘, “, 與 ` 這些符號在變數定義中的用途？

雙引號內的特殊字元如 $ 等，可以保有原本的特性（显示变量具体的值）
單引號內的特殊字元則僅為一般字元（变量还是显示$home这类）
那個 ` 是鍵盤上方的數字鍵 1 左邊那個按鍵。在一串指令的執行中，還需要藉由其他額外的指令所提供的資訊時，可以使用反單引號

13.跳脫符號 \ 有什麼用途

长指令时可以下一行，\enter

14.連續命令中， ;, &&, || 有何不同？

；，结束，然后执行下一条指令，&&，||是判断，&&前一个正确，后一个执行，前一个错误，后一个不用执行，||则相反。

15.如何將 last 的結果中，獨立出帳號，並且印出曾經登入過的帳號？

last | cut -d ‘ ‘ -f 1 | uniq

16.請問 foo1 && foo2 | foo3 > foo4 ，這個指令串當中， foo1/foo2/foo3/foo4 是指令還是檔案？ 整串指令的意義為

foo1,foo2,foo3为指令，foo4为档案（文件）。意义：当(当foo1为真时，执行foo2)为假时执行 foo3指令 取代foo4文件中的内容。 foo1&&foo2为真是才可以通过管道|输出到foo3。

17.如何秀出在 /bin 底下任何以 a 為開頭的檔案檔名的詳細資料？

ls | grep a*

18.如何秀出 /bin 底下，檔名開頭不是 a-d 的檔案？

ls | grep -v a*

19.我想要讓終端機介面的登入提示字元修改成我自己喜好的模樣，應該要改哪裡？(filename)

/etc/issue

20.承上題，如果我是想要讓使用者登入後，才顯示歡迎訊息，又應該要改哪裡？

/etc/motd
